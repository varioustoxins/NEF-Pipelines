Tools for composing and running NEF Pipelines NMR data pipelines.

**Start Here:** read nef://readme then nef://skill on every new session.

**Typical Workflow:**
1. read nef://readme — high-level overview
2. read nef://skill — expert guidance
3. nef_list_commands() — command overview
4. nef_get_command_help(pattern) — detailed help
5. nef_execute_command() — prototype single steps
6. nef_execute_pipeline() — production multi-step pipelines
